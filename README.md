## Overview

Built on 12,276 rows and 123 features from the Oracle's Elixir 2024 competitive dataset, this project trains and evaluates four classifiers — KNN, SVM, MLP, and Logistic Regression — to predict match win/loss outcomes. The pipeline handles high missingness (up to 97% in some columns), team-vs-player row alignment, and mixed feature types across player-level and team-level statistics.

The project was built for Starfall Serpents Esports as part of ISM-6251 at the University of South Florida, with the goal of identifying which in-game factors most strongly predict victory.

## Results

| Model | Accuracy | F1 | ROC-AUC |
|---|---|---|---|
| KNN (k=11, Manhattan, distance-weighted) | 91.5% | 0.962 | 0.974 |
| SVM (RBF kernel, C=10, gamma=auto) | 99.5% | 0.995 | — |
| MLP (Adam, 96 units, ReLU) | 99.5% | 0.995 | 0.9998 |
| Logistic Regression (C=1, L1, liblinear) | 99.5% | 0.995 | — |

> **Note on data leakage:** The high accuracies for SVM, MLP, and Logistic Regression reflect data leakage — the feature set includes post-game statistics (final gold totals, tower counts, kill totals) that are only known after the match ends. A production-ready model would restrict features to those available at a defined prediction point such as the 15-minute snapshot columns. KNN's lower accuracy (91.5%) is still inflated for the same reason.

## Pipeline

1. **Data loading** — reads directly from the committed `.xlsx` file using pandas + openpyxl
2. **Team→player backfill** — team-summary rows are merged back into player rows, then dropped
3. **Cleaning** — column normalization, duplicate removal, NA token standardization
4. **Feature selection** — identifier columns excluded; numeric and low-cardinality categorical columns retained
5. **Target creation** — binary `result` column coerced to `{0, 1}`
6. **Train/test split** — 80/20 stratified split
7. **Preprocessing pipeline** — median imputation + StandardScaler for numerics; most-frequent imputation + OneHotEncoder for categoricals
8. **Model training** — GridSearchCV / RandomizedSearchCV with StratifiedKFold for all four classifiers
9. **Evaluation** — accuracy, precision, recall, F1, ROC-AUC, confusion matrix, classification report

## Key Predictive Features (Logistic Regression)

- **Economy**: earned gold share, gold spent, gold per minute
- **Tempo**: team KPM, assists at 15 minutes, CS per minute
- **Objectives**: opponent inhibitors taken, enemy jungle pressure

## Dataset

League of Legends 2024 Competitive Game Dataset — Oracle's Elixir.
12,276 rows, 123 features covering kills, deaths, assists, gold differentials, objective control, and early-game stats. After dropping team-summary rows, 10,230 player rows remain for modeling.
